# Preparing
## Create docker image for posgres
To create docker image execute:
> cd docker-db\
> create-db-image.cmd
# Build project
To build project and create docker image execute:
> build-docker.cmd
# Run project under Docker 
To run application execute:
>docker-start.cmd

To stop project execute:
>docker-stop.cmd
